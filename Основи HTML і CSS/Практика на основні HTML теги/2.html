Що таке CMS

CMS - "система управління контентом" (рушій) - написана PHP-програмістами основа для сайту, за допомогою якої ви зможете керувати сайтом (додавати контент, змінювати пункти меню тощо), не знаючи HTML і CSS.

Однак, для того щоб зробити сайт за допомогою CMS, потрібні послуги і програміста, і дизайнера, і верстальника. І капіталовкладення.

Які бувають cms

Бувають різні системи управління контентом: для інтернет-магазинів, для блогів, для форумів тощо.

Приклади cms

Приклади популярних CMS: Joomla, WordPress (для блогів), PhpBB (для форумів).

CMS-ки бувають платні та безкоштовні.
