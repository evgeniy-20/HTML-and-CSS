PHP і JavaScript

Мови програмування PHP і JavaScript дають змогу зробити сайт динамічним, тобто таким, що реагує на дії користувача. Наприклад, можна зробити красиве меню, що випадає, або слайдер.

Види скриптів

Для цього пишуться скрипти (англ. script - "сценарій") - програми, що дають змогу реагувати на дії користувача. Скрипти бувають двох видів:

  • ті, які виконуються на сервері, а результат їхнього виконання приходить у браузер до користувача вже в готовому вигляді. Це скрипти, написані мовою PHP. На ній пишуться CMS-ки - системи управління контентом.
  • ті, які виконуються прямо в браузері користувача. Це скрипти, написані мовою JavaScript. Вони найчастіше використовуються для того, щоб зробити сторінку зручнішою і красивішою.
